# Economic-Excellence
Repo for Eric, Leo, Zuzia, and myself. Also shoutout Steve

Simple budgeting app with four main components: A dashboard that gives an overall snapshot of user's finanicals, a mortgage page that provides meainingful insight into the current housing market (and whether the user can afford a specified housing price), a protfolio tab that takes in  user protfolio and provides real time updates on the market, and a finacial score tab with an overall financial health.
